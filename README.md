> 这是一个由golang+vue编写的web IM应用服务端 📦📦📦

#### 简单的功能

   - [x] 支持微博登录
   - [x] 端对端消息推送、图片发送、表情包
   - [x] 语音功能
   - [ ] 视频功能
   - [x] 支持离线消息推送
   - [ ] 创建群聊
   - [ ] 群聊消息推送
   - [x] 离线消息推送功能
   - [x] 响应式的前端界面支持pc与h5
