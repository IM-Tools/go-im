## GoIm

> 这是一个由golang+vue编写的web IM应用服务端 📦📦📦


#### 简单的功能

   - [x] 支持微博登录
   - [x] 端对端消息推送
   - [x] 支持离线消息推送
   - [ ] 创建群聊
   - [ ] 群聊消息推送
   - [x] 消息提醒功能
   - [x] 好看舒适的前端界面
